Above are the screen shots of Flownix App for the reference.
- This app is made in XML and MVVM Architecture.
The application is built using XML and follows the MVVM architecture for clean separation of concerns and scalability.
- It leverages Firebase Firestore for storing boards, lists, cards, and other project metadata, while Firebase Storage is used for managing user profile images and attachments.
- Inspired by Trello, the app supports Kanban-style boards, task cards, and lists, enabling users to organize work efficiently.
- Implements drag-and-drop functionality to move cards across lists and boards, allowing seamless workflow management and real-time updates.
